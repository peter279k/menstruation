How can it be so beautiful yet responsive?

Don't believe us ? Take a look at the demo on the left. It resembles a standard ~220px WordPress, widget container size. The calendar fits perfectly to any size and works beautifully on touch devices like iPhones or Android devices.

Also, take note there are skins that are responsive. This ( skin-responsive-galileo ) and the Clean Skin pressented next ( skin-responsive ).

There are API calls too for reiniting the calendar with new events, changing the current date showed, and getting the clicked event data. Try set date to July 2015
